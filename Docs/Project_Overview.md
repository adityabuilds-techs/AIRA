# 🤖 AIRA

## Project Name

AIRA (AI Robotic Assistant)

## Purpose

AIRA is an AI-powered robotic assistant developed to perform
voice interaction, obstacle detection, camera vision,
and autonomous movement using Raspberry Pi 4 and ESP32.

## Goals

- Voice Assistant
- Face Recognition
- Camera Vision
- Obstacle Detection
- Touch Display
- AI Processing
- Autonomous Navigation

## Hardware

- Raspberry Pi 4
- ESP32
- Raspi  Camera
- Waveshare Display
- HC-SR04
- Speaker
- USB Microphone
- L298N
- TT Motors
- IR sensors
