# 🤖 AIRA

AIRA is an AI-powered robotic assistant being developed by Aditya.

## 🚀 Features

- Voice Assistant
- Face Recognition
- Obstacle Avoidance
- Camera Vision
- Raspberry Pi + ESP32 Communication
- Touch Display Interface
- AI Interaction

## 🛠 Hardware

- Raspberry Pi 4
- ESP32
- Raspi Camera Rev 1.3
- Raspi Display
- HC-SR04 Ultrasonic Sensor
- TT Motors
- L298N Motor Driver
- USB Microphone
- Speaker
- I.R sensors

🚧 Under Development

## 📌 Future Goals
